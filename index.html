Survey Note

This tool recommends preliminary spacing only. Actual supports shall be checked for line configuration, valves, thermal movement, vibration, nozzle loads, structures and all applicable design loads.

Calculation Basis

This trial tool uses the PIM-2 verified seed classes below and a preliminary steel pipe spacing basis. It estimates vertical distributed weight only. It does not calculate support member capacity, anchors, guides, thermal force, wind, seismic, water hammer, PSV thrust or equipment nozzle loads.

Core Logic

Total distributed weight w = pipe + fluid + insulation
Intermediate support reaction R = w × Lref
End support reaction R = w × Lref / 2

The calculated reaction is an estimated vertical distributed pipe weight at the reference spacing, not a final support design load.

PIM-2 Seed Classes Included

Only classes verified from the supplied Piping Material Classification source are included in this trial seed database. Additional classes can be added after database validation.

References & Limitations

  1. PIM-2 Piping Materials Classification, document IK-4006-5022 — service, material and schedule basis for included seed classes.
  2. MSS SP-58, Table 4 — preliminary maximum horizontal steel pipe hanger and support spacing basis.
  3. ASME B36.10M / B36.19M — nominal outside diameter and wall thickness data used for pipe mass calculation.

For final engineering use, verify the line class against the original PMC/PMS and perform the required piping/support/stress/structural checks.
